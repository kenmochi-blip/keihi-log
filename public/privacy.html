プライバシーポリシー

合同会社Smart&Smooth（以下「当社」）は、当社が提供するウェブアプリケーション「経費ログ」（以下「本アプリ」）における個人情報の取り扱いについて、以下の通り定めます。

1. 収集する情報

本アプリは、Googleアカウントによるログインに際し、以下の情報へのアクセス許可を求めます。

情報の種類用途
メールアドレスユーザーの識別・管理者権限の判定
Google スプレッドシート（アプリが指定したシートのみ）経費データの読み取り・書き込み
Google Drive（アプリが作成したファイルのみ）領収書画像のアップロード・取得

2. データの保存場所

本アプリに入力した経費データおよび領収書画像は、ユーザー自身のGoogleアカウント（Google スプレッドシートおよびGoogle Drive）にのみ保存されます。当社のサーバーに経費データや領収書画像が送信・保存されることはありません。

ライセンス認証のため、ライセンスキーのみを当社のサーバーに送信します。個人情報・経費情報はライセンス認証には使用されません。

3. センシティブデータの保護

本アプリはGoogleアカウントを通じた認証情報（アクセストークン）を含むセンシティブなデータを取り扱います。これらのデータを保護するため、以下の措置を講じています。

  • 通信の暗号化：本アプリと外部サービス間のすべての通信はTLS（HTTPS）により暗号化されます。
  • アクセストークンの非永続化：GoogleのOAuthアクセストークンはブラウザのメモリ内のみで保持し、当社サーバーへの送信や永続的なストレージへの保存は行いません。
  • 最小権限の原則：本アプリはアプリが作成または指定したGoogleスプレッドシートおよびDriveファイルにのみアクセスします。それ以外のGoogleアカウントのデータには一切アクセスしません。
  • データの非通過：経費データおよび領収書画像は当社サーバーを経由せず、ブラウザからGoogleのAPIに直接送信されます。
  • 領収書画像のAI解析：AI解析を利用する場合、領収書画像はユーザーが設定したGoogle Gemini API（Google LLC提供）に直接送信されます。当社は画像データを受信・保存しません。詳細は「5. AI機能について」をご参照ください。
  • ローカルストレージ：スプレッドシートIDなどの設定情報はブラウザのローカルストレージに保存されます。機密性の高い認証情報（トークン等）はローカルストレージに保存しません。

4. 情報の利用目的

  • 本アプリの機能提供（経費申請・一覧・集計・AI領収書解析）
  • ライセンスの有効性確認
  • アプリの不具合対応・改善

5. AI機能について

本アプリは、領収書の自動読み取り機能にサードパーティのAIサービスを使用します。

項目内容
AIサービス提供者Google LLC（Google Gemini API）
送信データユーザーがアップロードした領収書・レシートの画像
送信先Google Gemini API（generativelanguage.googleapis.com）
APIキーの管理チームプランでは管理者がGoogle AI Studioで取得したAPIキーをアプリに設定します。ソロプランでは個人が取得・設定します。APIキーはユーザー自身のGoogleスプレッドシート内に保存され、当社サーバーには保存されません。
当社によるデータの受信なし（画像データはブラウザから直接Google Gemini APIに送信）

Google Gemini APIによる画像データの取り扱いについては、Googleのプライバシーポリシーをご参照ください。

6. 第三者への提供

当社は、以下の場合を除き、収集した情報を第三者に提供しません。

  • ユーザー本人の同意がある場合
  • 法令に基づき開示が必要な場合

7. 利用するGoogleのOAuthスコープ

本アプリは以下のOAuthスコープのみを使用します。

スコープ用途
userinfo.emailログインユーザーのメールアドレス取得
https://www.googleapis.com/auth/spreadsheets経費スプレッドシートの読み書き
https://www.googleapis.com/auth/drive.fileアプリが作成したDriveファイル（領収書画像）へのアクセス

上記以外のGoogleアカウント情報にはアクセスしません。

7-2. Google API サービス利用規約・Limited Use 準拠声明

Google Workspace APIs から取得したユーザーデータ（生データおよびその派生データ）の使用および第三者への転送は、Google API サービス ユーザーデータポリシー（Limited Use 要件を含む）に準拠します。

"The use and transfer of raw or derived user data received from Google Workspace APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements."

8. Cookieおよびローカルストレージの使用

本アプリは、設定情報（スプレッドシートIDなど）の保存にブラウザのローカルストレージを使用します。これらの情報はユーザーのデバイス内にのみ保存され、当社サーバーには送信されません。

9. 情報の削除

Googleアカウントの「サードパーティアプリへのアクセス」から、本アプリのアクセス許可をいつでも取り消すことができます。アプリのデータはGoogle スプレッドシートおよびDriveに保存されているため、削除はユーザー自身がGoogle上で行ってください。
